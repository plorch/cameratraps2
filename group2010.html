PCAP 2010

Images and scans of 2011 vibi plots for use in placing camera traps and for revisiting PCAP plots. These are actually the 2015 data for most points, since these are updated versions of the 2010 points.

Camera trap sites

We will put images and the .pdf file associated with each PCAP site here for access from the field to help find plots and set up camera traps so they don't interfere with the PCAP sites.
