Focus on Wildlife -- Cleveland Metroparks

This is the repository for our newsletters.

We have deployed hundreds of wildlife cameras throughout this system in an effort to study this ecosystem without disturbing it. We have paired each of these cameras with a long-term vegetation monitoring plot, which will enable us to model human-wildlife-vegetation relationships with an unprecedented level of detail at multiple scales. These newsletters represent progress made identifying animals in the images from these cameras using a Zooniverse crowdsourcing site.
